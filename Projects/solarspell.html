Digital Library Management System (DLMS) - SolarSPELL

Invision Link

SolarSPELL is a solar powered digital library made for resource-constrained areas around the world. The SolarSPELL DLMS is an admin portal meant for managing this which is a completely manual process at the moment.

User Research

Who is the user?
SolarSPELL Librarian

What is their current process?

  • Manually approve submitted files in an excel sheet submitted by curators
  • Tag these files with appropriate metadata
  • Download these files on an SD card
  • Organise the files in appropriate folders
  • Pass on sd card to developer to add supporting software to bring up library
  • Manually check if the files will fit on the SD card
