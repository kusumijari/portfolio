Digital Library Management System (DLMS) - SolarSPELL

Invision Link

SolarSPELL is a solar powered digital library made for resource-constrained areas around the world. The SolarSPELL DLMS is an admin portal meant for managing this which is a completely manual process at the moment.

The following were the initial wireframes which acted like an initial user testing prorotype. It was also a springboard for conversation to clarify and build on the user needs.



Takeaways:
  • Supporting software can also be updated and the provision for that should be in the portal.
  • Specific items for which version control would be required was clarified.
  • The natural flow of organizing files as soon as they were approved was made clear.
